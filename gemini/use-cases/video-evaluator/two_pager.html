GenAI Video Evaluator

From Flawed AI Video to Flawless Renders

A Closed-Loop Prompt Engineering Pipeline for High-Fidelity Assets

The Problem: The Hallucination Gap

Generative video creators suffer from high failure rates (melting subjects, lighting drift). Traditional CV cannot detect semantic errors, forcing engineers into manual, unscalable Quality Control (QC) backlogs that bottleneck commercial rollouts.

The Solution: An Autonomous Remediation Pipeline

A 5-step serverless evaluator that breaks video audits into specialized Agent nodes. Instead of reporting a failure, it uses Gemini 3.1 Pro to rewrite the generating prompt with strict physics overrides, feeding them back to Google Veo 3.1.

Technology Stack

Built with enterprise-grade multimodal infrastructure:

Gemini 3.1 Pro Google Veo 3.1
Vite / React HTML5 Canvas
Tailwind CSS Vertex AI

Key Value

This architecture translates purely qualitative visual feedback (e.g., "clipping hands") into Quantitative Prompt Corrections. It minimizes rendering costs, reduces human intervention by 90%, and enables verified bulk-delivery of video assets.

How to Get Started - The 5-Step Implementation

Building the continuous feedback loop between Auditor and Artist

1

Prompt Tuning

Initialize Multi-Agent nodes with a "Negative Bias" weighting, forcing critical stance auditing and bypassing agreeable-model syndrome.

2

Frame Ingestion

Canvas extracts keyframes at a 800px bounding-box cap. This locks down visual details while maintaining an Optimal Token Budget.

3

Audit & Score

Parallel execute models using mode: 'ANY'. This forces structured JSON responses, populating a deterministic timecode.

4

Prompt Vectoring

Remediation logic aggregates original prompt + timecode failures to inject Negative Prompt Weights (e.g., "no clipping").

5

Auto-Regen

The repaired prompt is re-rendered on Veo 3.1 with a STYLE anchor, locking aesthetics while fixing underlying mechanics.
